# Phase 4 — Information Architecture (Homepage)

## Narrative arc
**Promise → Proof → Personality → Path.** A first-time visitor (high local intent, anxious about letting a stranger do gas work) is met with *who/what/where + reassurance* (promise), shown *credentials + services* (proof), introduced to *Jamie himself* (personality — the signature, contrarian move), reinforced with *reviews + real work + area fit*, and converted via *an easy call/WhatsApp* (path).

## Mandatory (not counted in the 12)
- **Navigation** — sticky header: logo, links (Services, Areas, About, Reviews, Contact), persistent **Call** + **WhatsApp** buttons, skip-to-content, current-page state, mobile drawer.
- **Footer** — NAP (Gas Safe 977838, phone, email, Burgess Hill–Mid Sussex), service + area links, IG link, legal links (Privacy/Terms/Cookies), auto-year.

## Rule checks
- **Composition variety:** shapes = SPLIT-OFFSET, H-STRIP, GRID-4, CENTERED, SPLIT-OFFSET, GRID-3, H-STRIP, GRID-3, LAYERED, SPLIT-OFFSET, STACKED, FULL-BLEED. **No SPLIT-50-50 at all** (>40% non-50-50 ✓). **No 3 adjacent identical** ✓. **FULL-BLEED + LAYERED + CENTERED all present** ✓.
- **Density:** B,B,D,B,B,D,B,D,B,D,D,B — **no 3 adjacent DENSE** ✓.
- **Signature:** exactly one — Module 5 (Meet Jamie) ✓.
- **Objection coverage:** all 7 Category-Brief objections mapped to modules ✓ (none escapes).

## Signature module (Module 5 — Meet Jamie)
A reverse SPLIT-OFFSET where Jamie's **full-res portrait** sits offset/overlapping into an ink-tinted panel, with a short first-person intro that **fades/slides up on scroll (Tier-2 motion)** and his **signature line "Prevention over cure"** set in display weight + the Gas Safe 977838 badge inline. It reads like a personal introduction, not a corporate "about us". This is the one composition/interaction a multi-engineer competitor structurally cannot copy — it's the homepage's memorable moment and the answer to the swap test.

## Variants (swap-ins, not additions)
- **Seasonal hero variant:** pre-winter (Aug–Nov) hero headline emphasises "book your service before winter"; cold-snap variant emphasises repairs. Same module, swapped copy/photo.
